# The Spider Programming Language Book

This directory contains the official mdBook documentation for Spider created by Kv Avinash Sarma.

The implementation is the source of truth. Chapters document the compiler,
runtime, CLI, package manager, corpus examples, tests, milestone notes, and
current limitations present in this repository.

Build locally with:

```powershell
mdbook build book
```

If `mdbook` is not installed:

```powershell
cargo install mdbook
```
